Edit steps

This is a page that gives the steps to follow to modify and add a post.

  1. GitHub Account
  2. DyConBlog Repository
  3. Folder description

1. GitHub Account and Deusto Organization

DyCon Blog is hosted on the GitHub web service (GitHub Pages) for them you need to create a GitHub account associated with your @deusto email.

In addition, it is necessary to belong to the GitHub organization to add content to DyCon Blog for this, contact the following email: djoroya@deusto.es

3. DyConBlog Repository

If you have your github account logged into your local computer you can clone the repository

$ git clone https://github.com/DeustoTech/DyCon-Blog.git

4. Edit post

All the posts are written in the markdown (.md) format. You can find a syntax guide on the following page: basis-syntax

For the best edition of the post, it is necessary to use a text editor that supports markdown, such as the Visual Studio Code program. Also in need to install a plugin that render markdown, like for example Markdown All in One

You can edit with this view:

Each post has in its url it contains the address of the folder where it is located.

### Background and motivation

We consider the initial-value problem for a Hamilton-Jacobi equation of the form

$$
\tag{HJ}
\left\{ \begin{array}{ll}
\partial_t u + H(D_x u) = 0, & \text{in} \ [0,T]\times \mathbb{R}^n, \\
u(0,x) = u_0(x), & \text{in} \ \mathbb{R}^n,
\end{array}\right.
$$

where $u_0\in C^{0,1}(\mathbb{R}^n)$ and the Hamiltonian $H: \mathbb{R}^n\rightarrow\mathbb{R}$ is 
assumed to satisfy the following hypotheses:

$$
\tag{2}
H\in C^2(\mathbb{R}^n), \quad H_{pp}(p) >0, \ \forall p\in\mathbb{R}^n, \quad \text{and} \quad 
\lim_{|p|\to\infty} \dfrac{H(p)}{|p|} = +\infty.

$$

Update Post

This whole project has been done with the help of the jekyll program. You must have jekyll installed to be able to upload content directly. Another option is to upload your changes using Github, and notify the administrator ( djoroya@deusto.es ) to compile your changes.

If you have jekyll installed you can update the web directly being in the main project folder, using the following command

jekyll build

To see a preview of the web you can use

jekyll server
